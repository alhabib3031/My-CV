## 🛠 المهارات والمؤهلات التقنية

### 💻 تطوير البرمجيات
- **لغات البرمجة:** `C#`
- **أطر العمل (Frameworks):** `ASP.NET Core`
- **المنهجيات:** أنماط التصميم المتقدمة (Design Patterns)، ومبادئ هندسة البرمجيات.

### 🏢 الأدوات المكتبية
- **حزمة مايكروسوفت أوفيس:** مستوى خبير (متخصص في `Word` و `PowerPoint`)

### 🌍 اللغات

- **اللغة الإنجليزية:** مستوى متوسط 🟡
